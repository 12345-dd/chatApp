# Chat App (React + Node.js)

A simplified ChatGPT-style web application with a responsive frontend and Node.js backend serving mock data. Users can start new chats, ask questions, see tabular answers, and maintain session history.  

## Features

- Start a new chat session  
- Ask questions and receive **tabular answers**    
- Session-based chat with history  
- Sidebar showing all created sessions   
- Fully responsive for desktop and mobile  
- No database required â€“ all data is mock/in-memory  

## Tech Stack

- **Frontend:** React, TailwindCSS, JavaScript  
- **Backend:** Node.js, Express  

## Installation

### 1. Clone the repository

git clone https://github.com/12345-dd/chatApp
cd chatApp

### 2. Install Backend Dependencies

cd backend 
npm install 
node app.js

Backend runs on http://localhost:4000

### 3. Install Frontend Dependencies 

cd frontend 
npm install
npm run dev

Frontend runs on http://localhost:5173
